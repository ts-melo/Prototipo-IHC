MODUX

Uma interface contextual para modelos de linguagem, criada para reduzir confusão, melhorar organização e incentivar interações mais conscientes com IA.

Sobre o Projeto

O MODUX propõe diferentes modos de interação com IA, adaptando a interface ao objetivo do usuário.

O projeto surgiu a partir de entrevistas com usuários frequentes de ferramentas como ChatGPT, Claude e Gemini, identificando problemas relacionados a contexto, organização, dependência e usabilidade.

Problemas Identificados

Histórico confuso

Conversas longas dificultam encontrar informações antigas.

Mesma interface para tudo

Estudar, programar e brainstorm utilizam o mesmo fluxo.

Dependência da IA

Usuários relataram dificuldade em realizar tarefas sem auxílio.

Perda de contexto

Sessões extensas se tornam difíceis de acompanhar.
